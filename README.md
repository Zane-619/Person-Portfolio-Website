## Laurence Dingle — Personal Portfolio

## Overview
A **responsive personal portfolio website** showcasing Laurence Dingle’s biography, technical skills, and projects. Designed with clean UI, accessibility, and mobile-friendly layouts.

## Tech Stack
- **HTML5** – Semantic structure  
- **CSS3** – Styling, dark/yellow theme, responsive design  
- **CSS Grid & Flexbox** – Layout for hero, skills, projects, contact  
- **Accessibility** – ARIA labels and semantic elements  

## Projects
- **About Me — Elective 3**: Personal portfolio demo  
- **CampusCheck**: Face recognition attendance system  
- **Data Management**: Basic CRUD application  

## Features
- Hero section with circular profile photo  
- Skills section with progress bars  
- Responsive projects grid with GitHub links  
- Contact form with social links
